# Cyclocartography

## Technique Profile

- Practice role: advanced location timing overlay
- Doctrine: locational astrology
- Birth-time sensitivity: required
- Status: derivable

## What Question This Answers

When do transiting angular patterns activate a place more strongly?

## Use When

- you already understand the natal location picture
- timing matters for a trip, move, launch, or temporary stay
- you want to overlay current activation onto natal locational potential

## Do Not Use When

- the base natal location work has not been done
- you are using it as a replacement for natal lines
- you are flattening MC/IC longitude comparisons and AS/DS curve comparisons into the same method

## How to Read It

1. Start with natal lines and relocated structure.
2. Compute lines for the relevant transit date.
3. Compare where transit emphasis reinforces or agitates the natal location picture.
4. Use the result as a timing overlay, not as a standalone map.

## What Not to Do

- Do not compare all line geometries as if they are equally simple.
- Do not ignore the natal condition of the transit planet.
- Do not promise precision if the timing window is broad.

## Current App Support

Derivable with repeated `lines` calls at natal and transit dates. A dedicated comparison tool does not currently exist.
