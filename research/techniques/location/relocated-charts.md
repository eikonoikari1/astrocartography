# Relocated Charts

## Technique Profile

- Practice role: core location deep-dive
- Doctrine: locational astrology
- Birth-time sensitivity: required
- Status: derivable

## What Question This Answers

How do the same natal planets redistribute through houses and angles in a new location?

## Use When

- you already have candidate locations
- a line map needs structural confirmation
- you want to know which life departments become emphasized in a place

## Do Not Use When

- you only need a first-pass location scan
- the birth time is uncertain
- you are using relocated houses without checking line emphasis

## How to Read It

1. Keep natal planetary longitudes fixed.
2. Recompute angles and houses for the new location.
3. Check which natal planets become angular or change house meaningfully.
4. Compare the relocated chart back to the natal and locational line picture.

## What Not to Do

- Do not treat relocated charts as a substitute for ACG lines.
- Do not look only at one relocated placement without reading the full structural shift.
- Do not forget timing if the move is date-sensitive.

## Current App Support

Current CLI support is manual: `chart` for natal positions plus `houses` for the target location. A dedicated relocated-chart output would still be useful.
