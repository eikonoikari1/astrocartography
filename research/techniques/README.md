## How to Read This Index

- `Practice role` tells you how central the technique should be in a professional reading.
- `Doctrine` tells you which tradition it belongs to.
- `Birth-time sensitivity` tells you how quickly the technique breaks down when the input is weak.
- `Status` tells you whether the app implements it directly, supports it indirectly, or still needs new code.
