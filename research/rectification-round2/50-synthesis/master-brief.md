# Master Brief After First Delegated Wave

## What Changed
The first wave materially narrowed the problem.

The project now has four stronger claims than it did at the start:
- `±2 min` should be treated as a subset target, not a universal promise.
- Source provenance belongs inside the label model, not in a footnote next to it.
- The interview should be designed like a psychometric discrimination system, not a generic astrology consultation.
- The model should produce calibrated intervals with abstention, not forced exact answers.

## Strongest Findings

### 1. Minute-level feasibility is conditional
The identifiability work supports a high-separability subset defined by three intersecting conditions:
- trustworthy documentary source quality
- enough concrete life evidence
- chart geometry that actually changes inside narrow bins

This makes local separability a first-order research object. A simulation gate should come before any large modeling effort.

### 2. Data quality is part of the target
The data audit points to a tiered benchmark:
- a strict gold set for minute-level scoring
- a broader challenge set for calibration and abstention

Rounded times, provenance ambiguity, timezone uncertainty, and rectified labels mixed with documentary labels are structural problems, not cleanup details.

### 3. Adaptive QA remains promising, but only in a constrained form
The strongest interview path is no longer "ask about personality." It is:
- ask about life-structure splits
- use forced-choice and pairwise comparisons
- include contradiction probes
- keep appearance in a low-weight corroboration role

This sharply reduces the chance that the system simply rewards vague self-recognition.

### 4. Benchmark rigor is now clearer
The modeling pack is directionally strong:
- discrete posterior over time bins
- factorized evidence families
- strong non-astrological baselines
- calibration and abstention as mandatory outputs

This gives the project a credible evaluation spine.

## What Is Still Thin
- Which question families actually outperform others on real or simulated known-time cases.
- Whether Western and Vedic experts should stay separate or be fused.
- How to model fuzzy event dates without leaking information.
- How much signal appearance, public-biography, and optional multimodal features really add.
- Whether the high-separability subset is large enough to matter.

## Recommended Next Wave

### Wave 2A: Measurement And Benchmark Tightening
- Turn the separability simulation spec into a concrete benchmark protocol.
- Define the triage classifier for `high`, `medium`, and `low` separability cases.
- Formalize the gold-set audit checklist with exact exclusion logic.

### Wave 2B: Interview Validation Design
- Convert the item bank into scored question families with expected chart splits.
- Design contradiction probes and invalid-response checks as benchmarkable components.
- Specify how pairwise chart narratives are generated without becoming leading prompts.

### Wave 2C: Doctrine And Evidence Fusion
- Research whether separate Western and Vedic experts improve calibration.
- Research how to encode event-date fuzziness and evidence confidence in one posterior.

## Current Strategic Position
The project is in a much better state than it was before the delegated wave. It now has a viable architecture and a disciplined skepticism layer. The next wave should stop broad exploration and move toward measurable protocols.
