# Open Questions Registry

Questions are ranked by expected value of information.

## Tier 1

### Q1. When is `±2 min` even identifiable?
- What changes if answered: It determines whether the target is universal, subset-specific, or unrealistic.
- Current gap: We do not yet have a feature-sensitivity or separability map across the day.

### Q2. Which evidence families genuinely split nearby candidate charts?
- What changes if answered: It sets the evidence hierarchy and the interview design.
- Current gap: Practitioner opinion exists, but comparative evidence is thin.

### Q3. Can adaptive QA outperform static and random questioning?
- What changes if answered: It decides whether the interactive loop is core architecture or optional garnish.
- Current gap: Strong adjacent methods exist, but rectification-specific validation is missing.

### Q4. How noisy are high-quality birth-time labels in practice?
- What changes if answered: It shapes training targets, evaluation, and data-collection strategy.
- Current gap: We have source taxonomies, but not enough direct audit work.

### Q5. What baseline should astrology-informed models be required to beat?
- What changes if answered: It prevents false progress.
- Current gap: The baseline and negative-control plan is not yet specified.

## Tier 2

### Q6. Are pairwise chart comparisons a better interface than direct self-description?
- What changes if answered: It influences the whole QA protocol.
- Current gap: Good adjacent theory, little rectification-specific evidence.

### Q7. Which timing techniques stay useful when event dates are fuzzy?
- What changes if answered: It affects how the system handles month-level or season-level evidence.
- Current gap: Practitioner guidance is richer than tested comparative evidence.

### Q8. Does separating Western and Vedic experts improve calibration?
- What changes if answered: It changes architecture complexity and doctrine handling.
- Current gap: No comparative synthesis yet.

### Q9. What is the incremental value of appearance, presentation, voice, and public-biography signals?
- What changes if answered: It determines whether multimodal features are worth the complexity.
- Current gap: Evidence is weak and mixed.

### Q10. Which users or chart types are likely non-rectifiable from available evidence?
- What changes if answered: It sets abstention rules and product expectations.
- Current gap: No explicit taxonomy of hard or impossible cases yet.
