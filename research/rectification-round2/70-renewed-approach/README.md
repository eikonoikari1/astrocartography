# Renewed Approach

This subdirectory documents a stricter reset of the rectification research program.

The main change is simple:
- treat rectification as a full-day reverse-timing and separability problem first
- treat interview logic as secondary until it proves measurable value
- treat `±2 min` as a conditional capability, not a default ambition

## Summary

The earlier research produced a strong audit and benchmark framework, but it ran ahead of proof in a few areas. This reset narrows the project to the parts most likely to survive contact with real data:
- provenance-aware corpus construction
- deterministic full-day candidate scoring
- timing-technique behavior analysis
- strict non-astrological baselines
- benchmark-first iteration

The renewed approach delays or demotes:
- broad interview ambition
- multimodal inputs
- early doctrine fusion
- interface-heavy narrative logic

The immediate goal is no longer "build the whole rectification system."
It is "prove that meaningful separability exists in real open-source known-time corpora under strict controls."
