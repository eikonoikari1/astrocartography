# Technique Selection

This page answers a practical question: which technique should I use, and when should I stop adding more?

## Selection Sequence

```text
Client question
  -> natal description?
  -> timing?
  -> relocation?
  -> separate divinatory question?
       |
       v
Pick the narrowest technique family that answers the question
  -> check birth-time quality
  -> check doctrine fit
  -> add only supporting techniques that clarify the same question
```

## Use the Narrowest Valid Family

| Client question | Primary family | Supporting layers |
|---|---|---|
| "Who is this person?" | Natal delineation | lots, fixed stars, progressions only if the question is developmental |
| "When does this topic activate?" | Time-lord and timing methods | transits, solar return, progressions |
| "Where should I live, work, or travel?" | Location methods | relocated charts, timing overlay |
| "Will this specific thing happen regarding X?" | Horary | none unless the consultation is explicitly horary-led |

## Birth-Time Gate

| Birth-time quality | Safe techniques | Use carefully | Avoid |
|---|---|---|---|
| Exact | All supported techniques | None | None |
| Strong but not exact | transits, many natal techniques, some progressions | solar return, relocated charts | ZR, parans, local space if angle precision matters |
| Approximate | broad natal description, slow-planet transits | progressions with caution | house-based timing, relocation judgments, exact lot work |
| Unknown | sign-based natal themes, broad transits | very little else | all angle-heavy and house-dependent methods |

## What to Add, and What to Resist

Add a second technique when it answers one of these:
- It narrows timing.
- It checks whether the same topic is truly activated.
- It changes the place-specific recommendation.
- It reveals a conflict that needs resolution.

Do not add a second technique when:
- You are trying to rescue a weak birth time.
- The first technique already answered the question clearly.
- The second technique belongs to a different doctrinal lane and would only create noise.
- You want more confidence without a better evidentiary basis.

## When to Stop

Stop when you can name:
- the main topic
- the active planet or ruler
- the active time window or life chapter
- the location implication, if place is part of the question

If you cannot name those, add method. If you can, start interpreting.

## Separate Discipline Warning: Horary

Horary belongs in a different branch of practice.

- Use it when the client asks a focused, sincere, present-tense question.
- Do not treat it as a confirmation tool for natal or relocation work.
- Do not mix horary testimony into a multi-technique timing stack unless the entire consultation is built that way from the start.
